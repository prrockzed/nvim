Here I will keep all the keybindings specific to this configuration of Neovim.
The common keybinding are given all over the internet.
And if you are very tired of browsing the internet then just do: 
:Tutor

In Normal Mode:
<Space>     --> Which key setup
<Space><bs> --> Which key one level up
` or '      --> File which key
<C-c>       --> Copy whole file
W           --> Collapse nvim tree recursively
<C-v>       --> Selects vertically(Visual Block)
<S-v>       --> Selects Line(Visual Line)
<C-j>       --> Iterate to next iteration in neorg
fa          --> Format on save in neorg
<CR>        --> Hop to the link
